Project Information:

As for the class project, I would prefer to employ the "Online Marketplace for Buying and Selling" topic. I feel that this project topic is most closely aligned with what I may want to do with web development in the future. The use of online media for business transactions is obviously very relavent to today's globalized technological economy. In addition to the relavence of this topic, I also feel that the implementation of this project type would be the most fun for me. I really enjoy the frontend design aspect of web development. It allows me to express my creativity through code.

I have a few ideas in mind for this project. First, I was considering the implementation of a "Tinder for Sports" (the name obviously needs a little workshopping). Essentially, this would be an online platform for people within a reasonable area to network for fellow sports players. The platform would allow people to form teams for team sports, specify the location preference, and even book rooms/equipment at local rec centers directly on the site. An additional idea I had was a mediary marketplace for food purchasing. I know this has been done before (UberEats, Golo, etc.), however, it may be interesting to implement an eco-friendly version of these sites. This marketplace could implement eco-friendly practices by providing recyclable/compostable materials for each restaurant, promoting eco-friendly establishments, and providing information/fun facts pertaining to sustainability and eco-friendly nutrition along each step of the user's experience.
